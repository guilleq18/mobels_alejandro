# 🔐 ACCESO SEGURO A TUS CLAVES SSH

Tus claves están en:
- **Privada:** C:\Users\willi\.ssh\hostinger_key
- **Pública:** C:\Users\willi\.ssh\hostinger_key.pub

## OPCIÓN 1: Abrir directamente en Notepad (RECOMENDADO)

```powershell
# Abre la clave PRIVADA en Notepad
notepad $env:USERPROFILE\.ssh\hostinger_key

# Abre la clave PÚBLICA en Notepad
notepad $env:USERPROFILE\.ssh\hostinger_key.pub
```

Luego:
1. Ctrl+A para seleccionar todo
2. Ctrl+C para copiar
3. Pégalo donde necesites (GitHub Secret o Hostinger)

## OPCIÓN 2: Copiar directamente a portapapeles

```powershell
# Copiar clave PRIVADA
Get-Content $env:USERPROFILE\.ssh\hostinger_key | Set-Clipboard

# Copiar clave PÚBLICA
Get-Content $env:USERPROFILE\.ssh\hostinger_key.pub | Set-Clipboard
```

Luego pega donde necesites (Ctrl+V)

## OPCIÓN 3: Ver en terminal (para verificar)

```powershell
# Ver clave PRIVADA
Get-Content $env:USERPROFILE\.ssh\hostinger_key

# Ver clave PÚBLICA
Get-Content $env:USERPROFILE\.ssh\hostinger_key.pub
```
