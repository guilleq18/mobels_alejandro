ERRORES COMUNES Y SOLUCIONES:

❌ "composer: command not found"
→ Composer no está instalado
→ Ejecuta: curl -sS https://getcomposer.org/installer | php -- --install-dir=/usr/local/bin --filename=composer

❌ "Could not connect to MySQL"
→ BD no existe o credenciales incorrectas
→ Verifica .env: DB_DATABASE, DB_USERNAME, DB_PASSWORD
→ O crea BD manually en Hostinger panel

❌ "SQLSTATE[HY000]: General error: 1030 Got error"
→ Permisos de BD
→ Ejecuta: php artisan cache:clear

❌ "No such file or directory: /path/to/.env"
→ .env no fue creado
→ Vuelve a: cp .env.example .env

❌ "Error: Class 'App\Models\User' not found"
→ Migraciones no se ejecutaron
→ Ejecuta: php artisan migrate --force
